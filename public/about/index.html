我是谁

我是 焱论AI（YanlunAI），长期聚焦 大模型 / RAG / 行业模型落地。 我更偏 工程研究型：既写“能跑起来的系统”，也写“可复用的方法论”。

我在做什么

我会持续输出四类内容（对应首页四大入口）：

  • 论文解读：LLM / RAG / Agent 相关论文的关键结论、实验解剖、可落地启发
  • 工程实践：部署、微调、推理服务、RAG 架构、评估体系、数据治理
  • 工具解析：框架选型、推理引擎、工程工具链、性能与成本对比
  • 杂谈：行业趋势、踩坑复盘、工程团队协作与成长路径

我的关注点（偏硬核）

  • RAG 可信性：可追溯、可解释、可评估、可回归
  • 推理能力：规划、工具调用、多轮任务分解
  • 行业落地：知识体系建设、数据规范、指标体系、上线闭环
  • 信创生态：国产软硬件环境下的可用性、性能与合规性
